**Image Classification Demo**

Интерактивное веб-приложение для демонстрации модели классификации изображений с визуализацией предсказаний и объяснимостью (Grad-CAM).

**Возможности**

- Загрузка изображений (JPG / PNG / JPEG);
- Отображение Top-3 предсказаний модели;
- Confidence scores для каждого класса;
- Grad-CAM визуализация (области внимания модели);
- Поддержка дополнительных сигналов модели (logits, если доступны);
- Отображение версии модели;
- 5 предзагруженных примеров для быстрого тестирования;
- Обработка ошибок inference без падения приложения.

**Структура проекта**

demo/
  ├── app.py          # Streamlit UI
  ├── models.py       # Inference (predict_one)
  ├── explain.py      # Grad-CAM
  └── examples/       # Тестовые изображения

**Установка зависимостей**

pip install streamlit pillow numpy

Если используется PyTorch:

pip install torch torchvision

**Запуск приложения**

streamlit run app.py

После запуска приложение будет доступно по адресу: http://localhost

**Интерфейс модели**

Приложение использует функцию: 

predict_one(image: PIL.Image) -> dict

**Формат ответа модели**

{
  "class": "...",
  "confidence": 0.75,
  "class_2": 0.24,
  "class_3": 0.02,
  "logits": [...]
}
