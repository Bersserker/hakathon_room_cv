# Release RC1

## Candidate

Current candidate: `cv03_balanced_sampler` ConvNeXt-Tiny fold ensemble.

## Files

- Release config: `configs/release/rc1.yaml`.
- Class mapping: `configs/data/class_mapping.yaml`.
- RC decision: `reports/rc1_decision.md`.
- Model card: `releases/rc1/model_card.md`.
- Checksums: `releases/rc1/sha256sums.txt`.

Checkpoints are referenced from `artifacts/checkpoints/` to avoid duplicating large files in the release directory.

## Generate submission

```bash
uv run python -m src.inference.predict --config configs/release/rc1.yaml
```

Output:

- `releases/rc1/submission.csv`.
- `releases/rc1/predictions.parquet`.

## Validate submission

```bash
uv run python -m src.inference.validate_submission \
  --submission releases/rc1/submission.csv \
  --test-csv data/raw/test_df.csv \
  --class-mapping configs/data/class_mapping.yaml
```

## Demo

```bash
uv run python demo/app.py --config configs/release/rc1.yaml
```

## Known caveat

`submission.csv` is generated by the inference command and is not committed by this bundle if inference has not been run on the local machine.
